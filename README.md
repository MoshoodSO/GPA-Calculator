## Welcome to GPA Genius!

> A smart GPA calculator application designed to help students calculate, track, and analyze their academic performance across semesters.

## Description

**GPA Genius** is a modern GPA calculation tool built using **Lovable.dev**, an AI-powered platform for creating full-stack web applications from natural language prompts.

The app helps students:
- Calculate GPA per semester
- Track cumulative GPA across multiple semesters
- Visualize academic trends
- Plan and project better academic outcomes

The project is designed for simplicity, accuracy, and ease of use, making it suitable for students at all academic levels.
